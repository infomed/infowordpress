Permite mostrar la lista de blogs, de una instancia de Wp Mu y sus usuarios con un rol específico.

La personalización de este bloque se basa en:

  • Título
  • Url de Wordpress Mu: seleccione la Url de Wordpress Mu.
  • Ver resultados:
    • Cantidad de Blogs: mayor cantidad de blogs a mostrar en el bloque.
    • Usuario por rol: seleccione si desea mostrar los usuarios del blog con un rol específico o no mostrar usuarios.
  • Región: se asigna el bloque a alguna región para que se muestre.
